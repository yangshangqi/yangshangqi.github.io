余数之和

数论, 约数

1.k % i = k - (k / i) * i, 所以我们要求的答案就是:n * k - (k / i) * i(1 <= i <= n)
2.令g(x) = k / (k / x),(1 <= x <= k)由夹逼准则可求得k / g(x) = k / x,所以对于 i 属于[x, k / (k / x)],k / i 的值都相同，等于k / x所以算式(k / i) * i = (k / x) * i,在区间[x, k / (k / x)]就是一个公差为(k / x)的等差数列，直接用等差数列求和公式计算即可

阶乘分解

数论, 素数

1.若把1至N中的每个数分别分解质因数,再把结果合并，时间复杂度过高为O(NlogN)
2.N!的质因子不会超过N，所以我们可以先筛选出1至N的每个质数p,然后考虑N！中有多少个质因子p
3.N!中质因子p的个数等于1至N每个数包含质因子p的个数之和。在1至N中p的倍数显然有n / p个，而p^2的倍数，即至少含有2个质因子p的个数有n / p^2个,因为其中的一个质因子已经在N / p中统计过了，所以这里不需要乘以2
4.一次类推，我们便可以得到N！中质因子p的个数为(N / P) + (N / P^2) + (N / P^3) + ….. + N / plogp(N)
