整数运算

1.如果俩个操作数有一个为long，则结果为long

2.如果没有long时，结果为int，即使操作符全为float或byte，也必须用int类型的变量来保存

浮点运算

3.如果俩个操作符中有一个为double，则结果为double

4.只有当俩个操作符全为float时，结果才为float

取模运算
