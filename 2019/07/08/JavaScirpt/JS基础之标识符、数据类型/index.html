JS基础之标识符、数据类型

2019-07-08

JavaScirpt

一.标识符

1.在JS中所有的可以由我们自主命名的都可以称为标识符

2.命名规则：

1.标识符中可以含有字母、数字、下划线和$。
2.标识符不能以数字开头
3.标识符不能是关键字

4.标识符一般采用驼峰命名法：

首字母小写，每个单词的开头字母大写，其余字母小写

如: var helloWorld = 123；

二.数据类型

数据类型指的是字面量的类型，一共有6种

String 字符串

Number 数值

Boolean 布尔值

Null 空值

Undefined 未定义

Object 对象

前5个为基本数据类型，Object属于引用数据类型

1.字符串

声明:

1>.单引号，双引号都可以，但必须配对，不能混淆

2>.引号不能嵌套，双引号里不能放双引号，单引号里不能放单引号

3>.双引号里可以嵌套单引号

4>.如果想要任意嵌套，必须加入转义字符

转义字符：

在字符串中可以用\作为转义字符

如\ “表示 单引号， \ n 表示换行， \ \ 表示斜杠

2.数值

1>.声明

2>.可以用 typeof 来检查一个变量的类型

3>.输出数字的最大值、最小值

4>.NAN 是一个特殊的数字， 表示Not A Number

5>.如果JS进行浮点元素，可能得到一个不确定的结果

3.布尔值 Boolean

4.Null

5.Undefined
