pycharm导入包问题

2019-12-09

使用pycharm安装模块的时候，经常出现安装不成功或超时的问题，

一般情况下pip出现ReadTimeoutError都是因为被GFW给墙了，所以一般遇到这种问题，我们可以选择国内的镜像来解决问题。

注：防火长城（英语：Great Firewall (of China)，常用简称：GFW，中文也称中国国家防火墙，中国大陆民众俗称墙、防火墙、功夫网等等），是对中华人民共和国政府在其互联网边界审查系统（包括相关行政审查系统）的统称。

解决方法：

一.使用镜像。

常用的有镜像有
清华大学镜像：https://pypi.tuna.tsinghua.edu.cn/simple/
阿里云：http://mirrors.aliyun.com/pypi/simple/
中科大镜像：https://pypi.mirrors.ustc.edu.cn/simple/
豆瓣镜像：http://pypi.douban.com/simple/
中科大镜像2：http://pypi.mirrors.ustc.edu.cn/simple/
使用镜像的安装范式为：pip install xxxx -i http://… 或者 pip install -i http://… xxxx。

示例如下：

pip install imageio -i https://pypi.douban.com/simple
